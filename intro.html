CSS - Cascading Style sheets

CSS is a programming language used to improve the presentation of a html page

CSS will help modifying aspects like text sizes , colours, background, image sizes and shapes, borders, positions, responsiveness(different looks on different screen sizes) among other aspects of presentation

Css is used to design user interfaces for web pages and it helps enhance user experience

Css syntax(how do we write Css)

A Selector is used to target an element or a group of elementsfor styling

{} - A declaration block- groups properties and values to be applied on the selector

Property- a presentation aspect you'd want to modify eg. colour, size, position

Value - the final look you are interested in for the page ,fullpage,400px ...

selector{ property: value; property: value; }

Example

hi{ color: green ; }

Where do we put our CSS

  • Inline CSS

    we use the style attribute on individual element. (Code is less reusable and makes maintaining the application difficult

  • Internal CSS

    Using the style element .(Doing styles within same files as a structure is not recomended. styles also won't be shared to other html pages.)

  • External CSS (Recomended)

    using a different file (.css) linked to the html page using the link element usualy placed in the head of a html page.
