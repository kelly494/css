CSS Selectors

CSS selectors are patterns used to select and style elements in a html document

  • Universal selector *

    Selects all elements in the page

  • Elements Selector / Type Selector

    Selects all elements of a given type e.g. p,img,h2,.....

  • Class Selector

    Selects a single element with a specific class

  • ID Selector

    Selects a single element with a specific ID

  • Grouping and combining selectors

      • Groups Selector

        Applies the same style to multiple selectors

      • Descendant selector (space)

        Selects only direct children of an element

      • Child selector (>)

        Selects only direct children of an element

      • Adjacent sibling selector (+)

        Selects the first adjacent sibling element.

      • General Sibling Selector (~)

        Selects all sibling after the specified element
