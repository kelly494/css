CSS Units & Measurements

CSS provides various units to define dimensions, spacing, and typography. These units are categorized into absolute and relative types.

1. Absolute Units

Absolute units define fixed sizes that do not change relative to the viewport or parentelements

  • px-pixels
  • cm-centimetre
  • mm-Millimetre
  • in- inches

2. Relative units--- Responsive design

Relative units depend on parent elements, viewport size,or root element, making them ideal fo responsive design

em and rem are font related units. em units is relative to the parent element font size and rem is relative to root element(html)font size

Percentage(%) units are relative to a parent element(width or height)

vw(viewport width) and vh(viewportheight) units are relative to the viewport/available screensize
